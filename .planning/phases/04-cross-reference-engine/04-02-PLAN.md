Tests use inline data construction — no fixture files needed. Helper methods build DiffResult and CoverageResult from dictionaries.

Key helper pattern:
```csharp
private static DiffResult MakeDiff(params (string path, int[] lines)[] entries) =>
    new(entries.ToDictionary(
        e => e.path,
        e => (IReadOnlyList<int>)e.lines.ToList(),
        StringComparer.OrdinalIgnoreCase));

private static CoverageResult MakeCoverage(params (string path, int[] lines)[] entries) =>
    new(entries.ToDictionary(
        e => e.path,
        e => (IReadOnlySet<int>)e.lines.ToHashSet(),
        StringComparer.OrdinalIgnoreCase));
```

Test scenarios to cover:

1. **Empty diff → Empty result**
   - Input: DiffResult.Empty, any CoverageResult
   - Expect: result equals CrossReferenceResult.Empty (Files.Count == 0, TotalAddedLines == 0)

2. **Exact path match — fully covered**
   - Diff: `"src/Foo.cs"` adds lines [10, 11, 12]
   - Coverage: `"src/Foo.cs"` has lines {10, 11, 12}
   - Expect: UncoveredLines is empty, TotalUncoveredLines == 0, UncoveredPercent == 0.0

3. **Exact path match — fully uncovered**
   - Diff: `"src/Foo.cs"` adds lines [10, 11, 12]
   - Coverage: `"src/Foo.cs"` has lines {5, 6} (no overlap)
   - Expect: UncoveredLines == [10, 11, 12], TotalUncoveredLines == 3

4. **Exact path match — partially covered**
   - Diff: `"src/Foo.cs"` adds lines [10, 11, 12]
   - Coverage: `"src/Foo.cs"` has lines {10, 12} (11 is uncovered)
   - Expect: UncoveredLines == [11], TotalUncoveredLines == 1

5. **Suffix match — coverage has absolute path prefix**
   - Diff: `"src/Foo.cs"` adds lines [5, 6]
   - Coverage: `"C:/repo/src/Foo.cs"` has lines {5, 6}
   - Expect: Both lines covered, TotalUncoveredLines == 0

6. **File absent from coverage — all lines uncovered**
   - Diff: `"src/Bar.cs"` adds lines [1, 2, 3]
   - Coverage: empty (or contains only unrelated files)
   - Expect: UncoveredLines == [1, 2, 3]

7. **ALL diff files appear in result — including fully covered ones**
   - Diff: two files — `"src/A.cs"` (covered), `"src/B.cs"` (absent from coverage)
   - Expect: result.Files.Count == 2; A.cs has empty UncoveredLines; B.cs has all lines uncovered

8. **Aggregate stats — multiple files, partial coverage**
   - Diff: `"src/A.cs"` adds [1,2,3], `"src/B.cs"` adds [10,11]
   - Coverage: A.cs has {1,3} covered; B.cs absent
   - Expect: TotalAddedLines == 5, TotalUncoveredLines == 3, UncoveredPercent ≈ 60.0

9. **Case-insensitive path matching**
   - Diff: `"src/Foo.cs"` (lowercase)
   - Coverage: `"SRC/FOO.CS"` (uppercase)
   - Expect: lines matched correctly
